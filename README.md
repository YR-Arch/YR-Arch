### 🔧 正在开发

| 项目 | 说明 |
|------|------|
| **控规上网数据质检软件** | 面向控制性详细规划成果的数据质量检查工具 — 自动校验空间拓扑、属性完整性、规范合规性 |
| **CAD ↔ GDB 双向转换器** | AutoCAD 图形与 Geodatabase 之间的无损双向互转，保留属性、图层结构和坐标体系 |
| **产业研究Agent引擎** | 产业链研究与投资分析的智能化引擎 — 市场规模测算、敏感性分析、情景推演，从数据到报告一键生成 |
| **规划咨询知识中枢** | 融合GIS分析与政策语义的规划辅助决策系统 — 用地适宜性评价、规划冲突检测、合规性审查 |
| **多 Agent 研究分析团队** | 编排者驱动的自动化研究与报告生产管线 — 从数据采集、统计分析到文档生成全链路 |
